**backend/app/services/tickets.py — business logic layer:**

`create_ticket(db, data: TicketCreate, actor_id: UUID) -> Ticket`:
- Creates Ticket with status_column=Backlog, owner_id=None
- Creates initial ColumnHistory row (ticket_id, column="Backlog", entered_at=now(), exited_at=None)
- Emits TicketEvent(event_type="created", payload={"title": data.title}, actor_id=actor_id)
- All three operations in single `async with db.begin()` transaction
- Returns the created Ticket

`move_ticket(db, ticket_id: UUID, data: TicketMoveRequest, actor_id: UUID) -> Ticket`:
- Fetches ticket; 404 if not found
- TICKET-07: if data.target_column == StatusColumn.Backlog and data.owner_id is not None → raise HTTPException(400, "Backlog tickets cannot have an owner")
- TICKET-08: if ticket.status_column == StatusColumn.Backlog and data.target_column != StatusColumn.Backlog and data.owner_id is None → raise HTTPException(400, "owner_id required when moving out of Backlog")
- Close open ColumnHistory row: `SELECT ... WHERE ticket_id=ticket_id AND exited_at IS NULL` → set exited_at = now()
- Open new ColumnHistory row: INSERT with column=data.target_column, entered_at=now(), exited_at=None
- Emit TicketEvent(event_type="moved", payload={"from": str(ticket.status_column), "to": str(data.target_column)}, actor_id=actor_id)
- If data.owner_id is not None and data.owner_id != ticket.owner_id: set ticket.owner_id, emit TicketEvent(event_type="assigned", payload={"owner_id": str(data.owner_id)}, actor_id=actor_id)
- All in single transaction; return updated ticket

**backend/app/routers/tickets.py:**
- All routes use `Depends(get_current_user)` — any authenticated user can create/edit (TICKET-03)
- `POST /api/tickets` → `create_ticket()` service; returns TicketOut 201
- `GET /api/tickets/{id}` → fetch ticket with selectinload(owner, department); 404 if missing; also selectinload ColumnHistory and TicketEvent for detail view; compute time_in_column from open ColumnHistory row; return TicketOut
- `PATCH /api/tickets/{id}` → accepts TicketUpdate; update non-None fields; emit TicketEvent(event_type="edited", payload={"fields": list of changed fields}); return TicketOut
- `DELETE /api/tickets/{id}` → `Depends(require_admin)` (TICKET-04); delete ticket (CASCADE deletes history + events); 204
- `PATCH /api/tickets/{id}/move` → calls `move_ticket()` service; returns TicketOut
- `GET /api/tickets/{id}/events` → return list[TicketEventOut] ordered by created_at asc (DETAIL-05)
- `GET /api/tickets/{id}/history` → return list[ColumnHistoryOut] ordered by entered_at asc; compute time_spent for each closed row (DETAIL-06)
