Follow the exact formula from RESEARCH.md Pattern 1:
  - weekly_cost = hours_per_week * employees * avg_hourly_cost (None if any input is None)
  - yearly_cost = weekly_cost * 52
  - annual_savings = yearly_cost * expected_savings_rate
  - dev_cost = effort_estimate * ai_team_hourly_rate
  - roi = (annual_savings - dev_cost) / dev_cost — ONLY if dev_cost is not None and dev_cost != 0; else None (ROI-05 guard)
  - adjusted_roi = roi * (1 - risk_probability) if roi and risk_probability are not None
