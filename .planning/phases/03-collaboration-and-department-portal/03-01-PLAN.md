**backend/app/models/ticket_comment.py** — TicketComment model:

- author_id: UUID FK → users.id
- body: Text NOT NULL
- created_at: DateTime(timezone=True) server_default=func.now()
- ticket = relationship("Ticket", back_populates="comments", lazy="raise")

**backend/app/models/ticket_subtask.py** — TicketSubtask model:
- id: UUID PK
- ticket_id: UUID FK → tickets.id ON DELETE CASCADE
- title: String(500) NOT NULL
- done: Boolean default=False server_default="false"
- position: Integer NOT NULL
- ticket = relationship("Ticket", back_populates="subtasks", lazy="raise")

**backend/app/models/ticket_template.py** — TicketTemplate model:
- id: UUID PK
- title: String(500) NOT NULL
- problem_statement: JSONB nullable (Tiptap JSON)

- default_next_step: Text nullable

**backend/app/routers/comments.py** — nested router prefix="/tickets/{ticket_id}/comments":
- POST / — create comment (body required, author=current_user, emit TicketEvent type="comment_added")
- GET / — list comments for ticket ordered by created_at ASC; use selectinload(TicketComment.author) to avoid lazy load
- DELETE /{comment_id} — author or admin delete; 403 if neither; 404 if not found; confirm ticket_id matches

Author/admin guard (inline, no new dependency): `if comment.author_id != current_user.id and current_user.role != "admin": raise HTTPException(403)`

**backend/app/routers/subtasks.py** — nested router prefix="/tickets/{ticket_id}/subtasks":
- POST / — create subtask; position = max(existing positions) + 1 (or 0 if none); return SubtaskOut
- PATCH /{subtask_id} — toggle done boolean; return SubtaskOut
- DELETE /{subtask_id} — delete subtask, then reassign positions 0..N-1 to remaining subtasks ordered by current position (fixes Pitfall 2 from RESEARCH.md)
- PATCH /reorder — accepts SubtaskReorderRequest (ordered_ids: list[UUID]); validates all IDs belong to ticket; assigns position = index; commits; returns list[SubtaskOut] sorted by new position

**backend/app/routers/templates.py** — router prefix="/templates":
- GET / — list all templates, selectinload created_by, return list[TemplateOut]
- POST / — create template, created_by_id = current_user.id, return TemplateOut 201
- PATCH /{template_id} — partial update any TemplateUpdate fields; 404 if not found; admin or member can update (any authenticated user)
- DELETE /{template_id} — delete; admin or member; 204

**backend/app/main.py** — add three include_router calls:
```python
app.include_router(comments.router, prefix="/api")
app.include_router(subtasks.router, prefix="/api")
app.include_router(templates.router, prefix="/api")
```
